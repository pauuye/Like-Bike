# Like Bike Sales Data Cleaning & Analysis (Excel)

## Overview
This project focuses on cleaning, transforming, and analysing a bike sales dataset using Microsoft Excel. 
It demonstrates practical data preparation and exploratory data analysis (EDA) techniques using spreadsheet tools.

## Key Steps

### 1. Duplicate Handling
- Identified duplicate records using Excel functions and filters
- Removed duplicate entries to ensure data accuracy

### 2. Data Cleaning
- Removed incomplete or inconsistent records
- Standardised formats (dates, categories, and numeric fields)
- Ensured dataset consistency before analysis

### 3. Feature Engineering
- Created calculated columns for key metrics:
  - Revenue
  - Cost
  - Profit

### 4. Exploratory Data Analysis

#### Customer Insights
- Analysed customer distribution by age group
- Evaluated average order quantity across:
  - Country
  - Product sub-category
  - Age group

#### Revenue Analysis
- Identified top-performing countries by average yearly revenue
- Analysed monthly revenue trends using pivot tables

#### Product Performance
- Analysed profit contribution by category
- Identified top sub-categories based on revenue
- Evaluated most profitable categories over time

## Key Insights
- Bicycles contribute the majority of total profit
- Road Bikes and Mountain Bikes are the top-performing sub-categories
- Expansion into accessories and clothing appears in later periods

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Functions (e.g. IF, VLOOKUP/XLOOKUP, COUNTIFS)

## Project Structure
- Raw Dataset
- Cleaned Dataset
- Analysis Sheets (Pivot Tables & Summaries)
