## Key Improvements Made

- Removed duplicate records using window functions
- Excluded incomplete data (2014–2016)
- Recalculated revenue, cost, and profit for accuracy
---
## Key Insights

- Bicycles generate the majority of profit (~77%)
- Road Bikes and Mountain Bikes are top-performing products
- Business expanded product categories over time
- Revenue trends show strong seasonal patterns
